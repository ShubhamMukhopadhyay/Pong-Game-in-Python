# 🏓 Pong Game in Python (Turtle Graphics)

A fully functional implementation of the classic **Pong Game** using Python’s built-in `turtle` module. This project showcases core programming concepts like **Object-Oriented Programming (OOP)**, **collision detection**, and **real-time game loops**.

---

## 🎮 Gameplay Overview

* Two players control paddles on opposite sides of the screen
* The ball moves continuously and bounces off walls and paddles
* If a player misses the ball, the opponent scores a point
* The game continues indefinitely with increasing ball speed

---

## 🧠 Concepts Used

* Object-Oriented Programming (Classes & Methods)
* Event Handling (Keyboard Controls)
* Collision Detection
* Animation using Game Loop
* State Management (Score tracking)

---

## 📂 Project Structure

```
├── main.py          # Main game loop and setup
├── paddle.py       # Paddle class (player controls)
├── ball.py         # Ball movement and physics
├── scoreboard.py   # Score tracking and display
```

---

## ⚙️ Features

* 🎯 Smooth paddle movement
* ⚡ Increasing ball speed after collisions
* 🧱 Wall and paddle collision detection
* 📊 Real-time scoreboard
* 🔁 Ball reset after each point

---

## 🕹️ Controls

| Player       | Key Bindings     |
| ------------ | ---------------- |
| Right Paddle | ↑ (Up), ↓ (Down) |
| Left Paddle  | W (Up), S (Down) |
